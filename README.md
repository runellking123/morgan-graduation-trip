# Dr. Morgan King — Doctoral Graduation Trip

Trip planning site for Morgan King's doctoral graduation from Capella University.

**Live Site:** [https://morgan-graduation-trip.vercel.app](https://morgan-graduation-trip.vercel.app)

## Pages

| File | Contents |
|------|----------|
| `index.html` | Hero, trip summary, links to topic pages |
| `travel.html` | Airbnb listings, Sprinter rentals, budget |
| `schedule.html` | Graduation logistics, day-by-day agenda |
| `explore.html` | Interactive Eat & Explore (filters, favorites, modal) |
| `explore.css` | Styles for the Explore page only |
| `explore.js` | Vanilla JS for Explore (localStorage `morganTripPlaces`) |
| `checklist.html` | Planning checklist |
| `styles.css` | Shared styles |

Shared assets: `morgan-photo.jpg` (and optional unused hi-res image in repo).
